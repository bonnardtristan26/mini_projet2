## 👋 Hey, Bienvenue dans **LaDiscorde**

**LaDiscorde** est l'un de nos plus grands projets — né avec une ambition claire :
devenir un **géant de la communication libre**, capable de **défier les plateformes capitalistes dominantes** et de redonner aux utilisateurs le contrôle total de leurs échanges.

LaDiscorde est une **plateforme de communication open-source** dont l'objectif est simple :
créer un espace **puissant, ouvert et accessible** pour discuter, collaborer et construire des communautés — sans compromis sur la liberté ni sur la vie privée.

## 🚀 Lancer le projet

```bash
# Cloner le dépôt
git clone https://github.com/ton-pseudo/LaDiscorde.git
cd LaDiscorde

# Installer les dépendances frontend
cd frontend && npm install

# Installer les dépendances backend
cd ../backend && pip install -r requirements.txt

# Lancer l'application
docker-compose up --build
```
